# RandomPasswordGenerator
# 🔐 Password Generator Web App

A modern and secure **Password Generator Web Application** built using **HTML, CSS, and JavaScript**.
This tool allows users to generate strong random passwords with customizable options such as length, numbers, uppercase letters, and symbols.

---

## 🚀 Features

* Generate **secure random passwords**
* Adjustable **password length**
* Options to include:

  * Uppercase letters
  * Numbers
  * Symbols
* **Copy to clipboard** functionality
* Clean and modern **glassmorphism UI**
* **Responsive design** for desktop and mobile
* Smooth button animations and user-friendly interface

---

## 🛠️ Technologies Used

* **HTML5** – Structure of the application
* **CSS3** – Styling and layout (modern UI design)
* **JavaScript (Vanilla JS)** – Password generation logic and interactivity

## ⚙️ How It Works

1. User selects password options:

   * Password length
   * Uppercase letters
   * Numbers
   * Symbols

2. When the **Generate Password** button is clicked:

   * JavaScript creates a pool of characters based on selected options.
   * Random characters are selected using `Math.random()`.

3. The generated password is displayed on the screen.

4. User can click **Copy** to copy the password to the clipboard.
